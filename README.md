## Before You Start

All scripts require PowerShell and Azure modules. [→ Setup Instructions](docs/SETUP.md)

## How to Use a Script

1. Choose a script from the table above
2. Read the script's README for overview
3. Check [USAGE.md](example/script/USAGE.md) for parameters
4. Copy a ready-to-use example from [EXAMPLES.md](example/script/EXAMPLES.md)
5. Run and export results to CSV
