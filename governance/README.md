# Governance Scripts

Audit and analyze Azure policy assignments, enforcement modes, and compliance status across your organization.

## Scripts in This Category

| Script | Purpose | Output |
|--------|---------|--------|
| [analyze-policy](analyze-policy/README.md) | Scan policy assignments across management group hierarchies; detect inheritance, enforcement mode, and managed identity usage | CSV with policy details + summary stats |

### When to Use

- **Audit compliance**: Understand which policies are assigned where
- **Inheritance review**: Identify inherited vs. directly assigned policies
- **Managed identity tracking**: Find policies using managed identities
- **Report generation**: Export policy landscape to stakeholders
